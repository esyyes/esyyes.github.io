  1. 按回车键输出消息，并不会在文本中表示回车

  2. 问题描述

    在按下回车键时，不仅会触发我们绑定的方法，还会触发回车键原本的换行效果，
    但我们希望只触发绑定的方法而不进行换行。

    解决方法

    在绑定事件函数的末尾加上 return ‘break’ 即可阻止按键的默认操作
