chatbot_版本1总结

检索式聊天机器人之旅游篇

数据：100条，数据匮乏，很多语句输出是错误的

流程：

  1. 对数据库中的数据进行spacy分词，还原，消除停靠词处理
  2. 利用规则语句输出一些简单的话
  3. 判断小于3个或者关键词不在数据中的，就输出sorry
  4. 利用tf-idf计算，然后将输入问题的关键词跟对应句子的进行得分计算
  5. 召回5个问题
  6. 计算相似度
  7. 利用自己写的余弦相似度和spacy自带的余弦相似度计算
  8. 输出相似度高于0.7且相似度最高的答案
