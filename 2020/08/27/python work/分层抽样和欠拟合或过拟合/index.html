过拟合和欠拟合问题

随机森林一般用更高的偏差去替换低的方差。因此只需要多加点决策树，去满足特征集合就可以，不需要再去考虑这个拟合的问题。

特征的确定

  1. 特征集合预处理：均值插补缺省值和zscore 标准化
  2. 利用随机森林获取特征重要度
  3. 然后筛选出重要度高于平均值的46个特征
  4. 再利用46个特征去获取平均重要度和得到这46个特征下特征递归算法，去获取特征的准确率
  5. 所以现在就是平均特征重要度确定后，用了一下PCA，重要的就17个啥的就可以了
